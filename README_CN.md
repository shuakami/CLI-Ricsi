## 🍁 安装步骤
1. 打开终端
2. 安装pnpm

   ```bash
   npm install -g pnpm

3. 安装Cli-Ricsi
   ```bash
   pnpm add cli-ricsi
4. 运行Cli-Ricsi
   ```bash
   pnpm clir

## 🤺 关于语言配置
您可以在以下路径找到语言配置文件：

>...\node_modules\cli-ricsi\lan.json
  ```bash
   {
  "language": "en" //编辑您想使用的语言
   }
  ```
